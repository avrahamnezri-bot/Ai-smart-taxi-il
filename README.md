# AI Smart Taxi System – מערכת מוניות חכמה

מערכת חכמה לניהול מוניות בישראל הכוללת פלטפורמה משולבת למנהל, נהגים ולקוחות:

## תיאור מערכת
המערכת מאפשרת:
- **דשבורד למנהל:** ניהול נהגים, משימות, רכבים, פרסום ותזמון.
- **אפליקציה לנהגים:** הרשמה, קבלת נסיעות, ניווט לכתובת הלקוח, עדכון סטטוס חי, תמיכה בשיחה קולית דרך Twilio Voice, והצגה גם בלוח הרכב.
- **אפליקציה ללקוחות:** הרשמה, הזמנת מונית, קבלת פרטי נהג בזמן אמת, מעקב אחר הגעת הרכב, הודעות והתראות.
- **AI חכם:** שילוב Grok AI, GPT-4, GPT-5 ו-Google AI, עם רוטציה שבועית/חודשית בין המודלים.
- **Twilio Voice:** תמיכה בשיחות רב-סיבוביות, עם זיהוי דיבור והפעלה חכמה של AI.
- **אתר להזמנות:** אתר אינטרנט ייצוגי עם פאנל למנהל לעיצוב ותמונות, הרשמה והזמנה ללקוחות, והצגת מידע לנהגים.
- **הרשאות מובייל:** ווידג'טים למיקרופון, רמקול, מיקום והתראות, כדי שהאפליקציות יתפקדו בצורה מלאה.
- **חיפוש והצגת מידע מהיר:** רוטציה בין מודלים כדי לשמור על דיוק וזמינות.

## התקנה ותלותיות
המערכת מבוססת Python עם Flask:
- Python 3.10+
- Flask
- Flask-Cors
- openai
- twilio
- sqlite3
- requests
